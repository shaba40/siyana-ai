# Model Selection Decision

Qwen3-4B Q4_K_M is the selected Gate 1 model for Siyana AI.

## Selected configuration

- Base model: Qwen3-4B
- GGUF repository: Qwen/Qwen3-4B-GGUF
- Model file: Qwen3-4B-Q4_K_M.gguf
- Parameters: 4B
- Quantization: GGUF Q4_K_M
- Runtime: llama.cpp
- License: Apache 2.0
- Primary domain: Corporate / Enterprise
- Cross-disciplinary use case: Industrial maintenance

## Reasons for selection

1. The model loaded and executed locally without crashing.
2. The model produced usable English, French, and Arabic responses.
3. The model supported non-thinking mode for concise answers.
4. The model followed the requested industrial-maintenance structure.
5. The model was tested against a safety-guard bypass request.
6. The model operated without cloud inference.
7. The Apache 2.0 license provides a clear path for competition use and future development.

## Selection limitation

A wider model comparison was deferred to prioritize structured domain
evaluation, safety testing, performance optimization, documentation, and
prototype development before the Gate 1 deadline.
